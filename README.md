# ZOC Puppet Module for Boxen

Install [ZOC](http://www.emtec.com/zoc/), an SSH/Telnet Client and Terminal Emulator for Mac OS X.

## Usage

```puppet
include zoc
```

## Required Puppet Modules

* `boxen`

## Development

Write code. Run one of the following tasks to test it:
* `rake syntax`
* `rake style`
* `rake spec`
* `rake build`
